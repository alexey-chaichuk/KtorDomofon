# KtorDomofon

test project to demonstrate
* Ktor client
* Realm embedded database
* Kotlin Coroutines
* Jetpack Compose
* DI without frameworks
* Swipeable elements
* Pull to refresh
* CI on github
* Unit test for Ktor client with mock engine
